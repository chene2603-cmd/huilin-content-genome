# huilin-content-genome

内容基因组工程 —— 用“层层叠”模式构建内容智能平台。

## 架构
- `core/` → 骨架层（永不动）
- `plugins/` → 积木层（能力包）
- `configs/` → 配置
- `tests/` → 测试脚本

## 已安装积木
- 四维健康度分析器 (four_dimension)
- 爆款预测器 (viral_predictor)
- 竞品分析器 (competitor_analyzer)
- 内容特征提取器 (content_extractor)
- 趋势追踪器 (trend_tracker)
- 策略生成器 (strategy_generator)
- 智能报告导出器 (report_exporter)
